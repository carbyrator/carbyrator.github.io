<h2>Проект 1. Task Tracker (TODO-приложение)</h2>
<p>Разработано приложение для управления задачами с поддержкой пользователей, авторизации и ролей. Реализован полный CRUD для пользователей и задач, статусы задач, приоритеты и сроки выполнения. Проект выполнен с использованием архитектуры MVC, шаблонов HTML (Jinja2) и базы данных SQLite с ORM и миграциями.
</p>
<ul>
  <li>Технологии: Python, Flask / Django / FastAPI, SQLite, ORM, HTML, MVC, Git.</li>
  <li>Ссылка: <a href="https://github.com/YannisBtw/task_tracker" target="_blank" rel="noopener">GitHub репозиторий</a></li>
</ul>

<h2>Проект 2. Оптимизация вычислений в Python</h2>
<p>Исследованы способы ускорения численного интегрирования методом прямоугольников. Реализованы версии функции с использованием потоков, процессов, Cython и отключения GIL. Выполнено профилирование, сравнительный анализ времени выполнения и тестирование корректности вычислений.
</p>
<ul>
  <li>Технологии: Python, threading, multiprocessing, Cython, timeit, unittest / pytest.</li>
