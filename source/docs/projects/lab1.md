<h2>Task Tracker (TODO-приложение)</h2>
<p>Разработано приложение для управления задачами с поддержкой пользователей, авторизации и ролей. Реализован полный CRUD для пользователей и задач, статусы задач, приоритеты и сроки выполнения. Проект выполнен с использованием архитектуры MVC, шаблонов HTML (Jinja2) и базы данных SQLite с ORM и миграциями.
</p>
<ul>
  <li>Технологии: Python, Flask / Django / FastAPI, SQLite, ORM, HTML, MVC, Git.</li>
