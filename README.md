# carbyrator.github.io
Портфолио. Я выбрал эту тему, так как она минималистичная и практичная: ничего не отвлекает от содержания, а все важные разделы (обо мне, лабораторные, контакты) находятся быстро. Такой дизайн хорошо подходит для портфолио разработчика, потому что делает акцент на проектах и результатах, а не на визуальных эффектах.
